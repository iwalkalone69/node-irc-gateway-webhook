# node-irc-gateway-webhook

## Description

A nodejs http server that posts webhook info into IRC.

## Project status

Platforms:
* GitHub
* GitLab
* BitBucket

Webhooks:
* Push
* Issues

## Installation

<code>npm install</code>

## Configuration

You can take a look at the configuration example config-sample.json

## Running

<code>npm start</code>

or

<code>node webhook.js</code>
